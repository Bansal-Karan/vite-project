# 🌐 Social Media App – Frontend

This is the frontend of the Social Media Application built using **React (Vite)**.  
It connects with the backend API to provide authentication, posts, comments, and user interactions.

---

## 🚀 Tech Stack

- React (Vite)
- JavaScript (ES6+)
- Tailwind CSS (if you used it)
- Axios / Fetch API
- React Router
- Zustand / Context API (if used for state management)

---

## ✨ Features

- 🔐 User Authentication (Login / Register)
- 📝 Create, Edit & Delete Posts
- ❤️ Like Posts
- 💬 Comment on Posts
- 👤 User Profile Display
- 🔄 Real-time UI Updates
- 📦 Secure API Integration with Backend

---

## 🔗 Backend Repository

This frontend connects to the backend API built using:

- Node.js
- Express.js
- MongoDB
- JWT Authentication

👉 Backend Repository Link:  
**https://github.com/Bansal-Karan/Socialmedia_backend**

Make sure the backend server is running before starting the frontend.
